## ⚙️ Understanding the Features

The application offers specific features to manage your output quality.

*   **Few-Step Distillation:** This core feature minimizes the number of passes the software takes to render an image. This leads to shorter wait times without losing significant detail.
*   **Distribution Matching:** This mathematical approach ensures the generated image matches the characteristics of the source style.
*   **Batch Processing:** You can add multiple prompts to a queue. The software processes these one by one, allowing you to walk away and return to a folder full of images.
*   **Export Options:** You can save your work as PNG or JPG files. Use the settings menu to toggle file formats or adjust image compression levels.

## 🛠️ Resolving Common Issues

Computers sometimes react in unexpected ways to new software. Use this list to troubleshoot common problems.

*   **The Software Will Not Start:** Ensure you have the latest NVIDIA drivers. If your drivers are outdated, the program might fail to initialize your graphics card. Visit the NVIDIA website to download the latest GeForce Experience or Studio drivers.
*   **Slow Generation Speed:** If the software runs slowly, it might be using your central processor instead of your graphics card. Check the "Device" menu in the settings to ensure your graphics card is selected.
*   **Out of Memory Error:** This happens when your graphics card lacks enough video memory to handle the requested image resolution. Try reducing the dimensions in the settings menu or close other programs that use the graphics card.
*   **Blank Images:** If you receive a blank file, check if your input prompt contains enough descriptive words. Very short prompts sometimes result in errors. Add more detail, such as style, lighting, or color, to improve results.
